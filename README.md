Machine Learning Tools Lab 🧠⚙️

Welcome to the Machine Learning Tools Lab repository! This collection contains practical assignments and experiments to understand and apply Machine Learning concepts using TensorFlow.js and Python. Each lab focuses on hands-on coding and real-world applications.

📌 Overview

Explore foundational ML concepts through experiments.

Implement models using TensorFlow.js in the browser.

Apply learning to real-world tasks like image classification, sentiment analysis, and gesture control.

🚀 Getting Started
Prerequisites

Node.js (optional, for running JS locally).

A modern browser (Chrome, Edge, Firefox).

TensorFlow.js loaded via CDN for browser-based experiments.

Run the Labs
git clone https://github.com/kanchan874/MLTL_Kanchan.git
cd MLTL_Kanchan
